To run the code, open the terminal and use command:

1. 
python main.py --part part1

for part 1 of PA2

2. 
python main.py --part part2

for part 2 of PA2

3. 
python main.py --part part3.1

for part 3 and the encoder portion of PA2

4. 
python main.py --part part3.2

for part 3 and the decoder portion of PA2
